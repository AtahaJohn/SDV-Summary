Frequently Asked Questions

FAQ

Where do I find the savegame?
Savegames are in %APPDATA%\StardewValley\Saves. This can be pasted into the file upload box to find the correct folder. Within this are subfolders for individual savegames. The file you should upload is named after your player (i.e. not named SaveGameInfo).
Can I upload multiple farms from the same playthrough?
Yes. Other uploads are automatically linked to below the farm name. You can also move between these farms using left and right keyboard keys.
I can't find my farm!
Re-upload your farm and the site will redirect you to the existing data, and re-set your ownership.
How do I enable savegame downloads?
If you own the page, beneath the minimap is a button to enable savegame downloads. This enables them for all users of the site and for all linked farms, and is irreversible unless all linked farms are deleted. In order to prevent another user from exploiting the upload mechanism and taking control of your uploads, you must set a passcode.
How can I delete my upload(s)?
When you upload a savegame a cookie is set to indicate ownership. Below the farm name should be a "delete this page" button. If you have uploaded multiple farms from the same playthrough you will also have an option to delete "all for this farm". Both ask for confirmation before deleting. If you have enabled savegame download, you will also need to input the passcode you set when you enabled this.
How do I use downloaded savegames?
Saves go in %APPDATA%\StardewValley\Saves where you can place the extracted folder and contents. This is normally
C:\Users\[username]\AppData\Roaming\StardewValley\Saves.
